# Prérequis 

* Avoir Java JDK 15 
* Avoir Maven
* Avoir un serveur tomcat pour déployer le war
* Récupérer le code de la branche voulu
* Générer le WAR de l'application
* Déployer le WAR sur le serveur tomcat
